# ⛪ Church Plant Toolkit

A cross-platform Flutter app designed to equip church planters with the tools and resources they need — all in one place.

Available on Android, iOS, Web, macOS, Windows, and Linux.

---

## 📖 About

Church planting is hard. This toolkit aims to reduce friction by giving church planters a practical, easy-to-use resource right on their device. Whether you're just starting out or already in the field, this app is built to serve you.

---

## ✨ Features

- 📚 Built-in dictionary of church planting terminology
- 📱 Cross-platform support (Android, iOS, Web, Desktop)
- 🔍 Searchable resources
- *(More features coming — contributions welcome!)*

## 🚀 Getting Started

### Prerequisites

- [Flutter SDK](https://docs.flutter.dev/get-started/install) (stable channel recommended)
- Dart SDK (comes with Flutter)
- A device or emulator to run the app

### Installation

```bash
# Clone the repository
git clone https://github.com/Malachi-Thompson/church_plant_toolkit.git

# Navigate into the project
cd church_plant_toolkit

# Install dependencies
flutter pub get

# Run the app
flutter run
```

### Running on a specific platform

```bash
flutter run -d chrome      # Web
flutter run -d android     # Android
flutter run -d ios         # iOS (macOS only)
flutter run -d macos       # macOS desktop
flutter run -d windows     # Windows desktop
flutter run -d linux       # Linux desktop
```

## 🔐 Security & API Keys

If you're building on top of this project and need to add API keys or secrets:

1. Copy `.env.example` to `.env` in the project root
2. Fill in your values
3. **Never commit your `.env` file** — it is already in `.gitignore`
